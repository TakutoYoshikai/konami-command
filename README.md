# konami-command
This is a bower package to register callback of konami command. konami command is "↑↑↓↓←→←→ba"

### Usage
**install**
```bash
bower install https://github.com/TakutoYoshikai/konami-command.git
```

**register callback**
```javascript
setKonamiCommand(() => {
  //Do something.
});
```
